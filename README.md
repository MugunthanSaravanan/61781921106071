
## Description
This project is designed to facilitate the registration of companies, obtain access tokens, and fetch products from a backend server using a React frontend.

---

## Setup
1. Clone the repository from [GitHub link].
2. Navigate to the project directory.
3. Install dependencies by running `npm install` in the terminal.
4. Start the backend server by running `node server.js`.
5. Start the React frontend by running `npm start`.
6. Access the application at `http://localhost:3000`.

---

## Files

1. **Register.js**:
   - Description: This file handles company registration to obtain client name and ID.
   - Usage: Execute `node Register.js` to register a company.

2. **Auth.js**:
   - Description: This file obtains the access code required for authentication.
   - Usage: Execute `node Auth.js` to get the access code.

3. **Server.js**:
   - Description: This file runs the backend server to serve API requests.
   - Usage: Execute `node Server.js` to start the server.

4. **React Frontend**:
   - Description: The React frontend fetches and displays products from the backend server.
   - Usage: Run `npm start` in the terminal to start the React application.

---

## Endpoints
- Company Registration: `POST /register`
- Authentication: `POST /auth`
- Product Listing: `GET /categories/:categoryname/products`

---

## Technologies Used

- React.js
